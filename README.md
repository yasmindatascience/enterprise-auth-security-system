# 🔐 Enterprise Authentication & Security System

## 📌 Visão Geral
Serviço central de autenticação e segurança utilizado por múltiplos sistemas corporativos.

---

## 🧱 Arquitetura

- JWT Authentication Service
- Refresh Token System
- OAuth2 (Google / GitHub)
- Role-Based Access Control (RBAC)
- Security Layer (Rate Limit + Brute Force Protection)

---

## 🔐 Funcionalidades

### Autenticação
- Login com email/senha
- Login social (Google / GitHub)
- JWT com expiração

### Segurança
- Proteção contra brute force
- Rate limiting por IP
- Hash seguro de senhas

### Autorização
- Controle por roles
- Permissões por recurso

---

## ⚙️ Tecnologias

- Python
- JWT (HS256)
- OAuth2
- SQL (PostgreSQL)
- Segurança de aplicações

---

## 🔗 Integração

Utilizado por todos os sistemas do ecossistema:

- Inventory System
